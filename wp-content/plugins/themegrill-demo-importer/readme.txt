Get free support at http://themegrill.com/support-forum/
Instruction on how to use http://themegrill.com/theme-instruction/importing-demo-content/

== Installation ==

1. Install the plugin by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the Maintenance Page Pro plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to the Settings->Maintenance Page Pro in your dashboard.
